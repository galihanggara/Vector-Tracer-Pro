# External Binaries

This directory contains the external command line executables required by Vector Tracer Pro.

## Required Binaries for Windows 10/11:

1.  **Potrace (`potrace.exe`)**:
    *   Download from the official Potrace SourceForge page: [https://potrace.sourceforge.net/](https://potrace.sourceforge.net/)
    *   Extract the ZIP package and copy `potrace.exe` to this `bin/` folder.

2.  **VTracer (`vtracer.exe`)**:
    *   Download from the Visioncortex VTracer GitHub repository Releases: [https://github.com/visioncortex/vtracer/releases](https://github.com/visioncortex/vtracer/releases)
    *   Download the Windows release binary, rename it to `vtracer.exe`, and copy it to this `bin/` folder.

*Note: Inkscape is not bundled with the application due to its size. Users should install Inkscape on their local machine, and the application will locate it automatically if it is available in the system PATH.*
